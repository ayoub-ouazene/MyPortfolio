# My Portfolio

A modern, animated portfolio built with **React + Vite**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## 📦 Build for Production

```bash
npm run build
```

Output goes to `dist/` — upload this folder anywhere.
